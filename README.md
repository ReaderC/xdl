# 🐦 X Media Downloader

> 一个简洁实用的 Chrome 扩展，用于下载 X（原 Twitter）上的高画质图片和视频

## ✨ 功能特性

### 🖼️ 图片下载
- 一键下载帖子中的**原始最高画质图片**（最高支持 4096×4096 PNG）
- 支持选择下载画质：4096×4096 / Large / Medium / Small
- 悬停在下载按钮上显示画质选择菜单

### 🎬 视频下载
- 自动获取并下载**最高码率** MP4 视频
- 支持 GIF 动图下载（以 MP4 格式保存）
- 采用多方案降级策略，确保下载成功率：
  1. 直接 MP4 URL（最快）
  2. Twitter Syndication API（无需登录）
  3. FixTweet 第三方 API
  4. Twitter GraphQL API（需要登录）
  5. M3U8 解析

### 👁️ 图片悬停预览
- 鼠标悬停图片一段时间后，自动弹出**全高度大图预览窗口**
- 预览窗口跟随鼠标左右移动，竖向始终占满视口高度
- 支持 **空格键** 固定预览窗口（固定后可拖动、滚轮缩放）
- 固定状态下支持 **← →** 方向键切换同推文多图
- **ESC 键**随时关闭预览
- 支持页面跳转/返回时自动关闭预览（兼容 X 的 SPA 路由）

### 📦 批量下载
- 一键抓取当前页面所有可见的图片、视频、GIF
- 可按媒体类型和画质筛选

### ⚙️ 设置面板
- 启用/禁用悬停预览
- 预览触发模式（自动 / 按住指定按键）
- 预览出现延迟（0 ~ 500ms 可调）
- 默认下载画质

## 📦 安装方法

> 本扩展尚未上架 Chrome 应用商店，需手动安装。

1. 点击页面右上角 **Code → Download ZIP**，解压到任意目录
2. 打开 Chrome，地址栏输入 `chrome://extensions`
3. 开启右上角 **开发者模式**
4. 点击 **加载已解压的扩展程序**，选择解压后的文件夹
5. 安装完成，刷新 X / Twitter 页面即可使用

## 🚀 使用方法

### 下载图片
鼠标悬停到帖子中的图片上，右下角会出现 ⬇️ 下载按钮，悬停按钮可选择画质，点击直接下载最高画质。

### 下载视频
鼠标悬停到帖子中的视频上，右下角会出现 ▶️ 下载按钮，点击后自动获取并下载最高码率 MP4。

### 悬停预览
鼠标停留在图片上约 300ms 后，旁边会弹出大图预览：
- **移动鼠标** → 预览窗口水平跟随
- **空格键** → 固定/取消固定预览窗口
- **← → 方向键**（固定状态）→ 切换同推文多张图片
- **ESC 键** → 关闭预览

### 批量下载
点击 Chrome 工具栏的扩展图标 → 选择媒体类型 → 点击 **批量下载当前页面媒体**。

## ⚠️ 注意事项

- 本扩展**仅支持** `x.com` 和 `twitter.com`
- 视频下载依赖 tweet ID，请确保帖子可公开访问
- 下载 4096×4096 画质图片时将转换为 PNG 格式
